# Somacord Typography

- **Headlines:** Lora (serif), weights 500–700
- **Body / UI:** Work Sans (sans-serif), weights 400–700
- **Sizing:** minimum body size 14px; headline sizes scale from 17px (labels) to 58px (hero)

See [design-system.md](../design/design-system.md) for how typography is applied within buttons, cards, and navigation.

Source: extracted from the Somacord design system build.
