## Usage
- **Deep Cord Blue** — nav, footer, primary headings, and dark sections
- **Community Green** — primary CTAs
- **Soft Sky / Warm Sand** — light section backgrounds
- **Charcoal** — dark text and dark sections (e.g. member stories)
- **White** — reverse logo only, and only on Deep Cord Blue backgrounds (never a colored-square logo block)

See [logo-guidelines.md](logo-guidelines.md) for logo-specific color rules, and [design-system.md](../design/design-system.md) for how color is applied across UI components.
