## Usage Rules
- Use the **primary (stacked)** or **horizontal** lockup as the default on light backgrounds; choose based on available width (horizontal for nav bars/headers, stacked for square or vertical placements).
- Use the **reverse** lockup only on Deep Cord Blue (#1F4E5F) backgrounds. Never place the reverse mark on a colored-square block.
- Use the **icon only** mark for small/constrained spaces (favicons, app icons, social avatars) where the wordmark would be illegible.
- Use the **wordmark only** where the icon is already established elsewhere on the same screen (e.g. repeated headers) or where a purely typographic lockup is needed.
- Maintain clear space around any lockup equivalent to the height of the "S" ligature.
- Do not recolor the mark outside the approved palette — see [colors.md](colors.md).
