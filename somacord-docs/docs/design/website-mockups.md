# Somacord Approved Website Mockups

Source file: `Somacord Experience.dc.html` (+ shared `EventCard.dc.html` component)

**Built prototype:** [`/assets/mockups/somacord-website-mockup.html`](../../assets/mockups/somacord-website-mockup.html) — a single-scroll HTML prototype covering Homepage, Gatherings, Gathering Detail, Speed Connect, Membership, Community Partners, City (Salt Lake City), and About, built with the approved design system and the [photography library](photography-direction.md).

## Pages
- **Homepage** — hero, gatherings showcase, how it works, membership preview, community partners teaser
- **Gatherings (discovery)** — featured Speed Connect, Community-created experiences, Partner experiences
- **Gathering Detail** — example "Coffee Meetup" with photos, description, time/location, join CTA
- **Speed Connect** — what to expect, why it's low pressure, what happens after
- **Membership** — example month, Founding Membership ($39/month) with benefit list, Community Partner cross-link
- **Community Partners** — partner types, partner benefits, one-membership messaging, become-a-partner CTA
- **City (Salt Lake City)** — example gatherings, local activity categories, the kinds of places we gather

All example content (gatherings, attendance ranges, city stats) is explicitly labeled "Example" in the UI and must be replaced with real data before public launch in a given city — see [/docs/product/mvp-requirements.md](../product/mvp-requirements.md#launch-honesty-rule).

See [app-mockups.md](app-mockups.md) for member-experience and mobile app screens, and [design-system.md](design-system.md) for the underlying component library.
