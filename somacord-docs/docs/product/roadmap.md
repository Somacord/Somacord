# Somacord Roadmap

- **Marketplace revenue** — featured placement, promoted events, ticket commissions, sponsorships, advertising. See [/docs/business/pricing.md](../business/pricing.md#future-marketplace-revenue).
- **Additional cities** beyond Salt Lake City — sitemap and SEO patterns already support this without code changes. See [/docs/website/sitemap.md](../website/sitemap.md).
- **Currently excluded from MVP**, candidates for later phases: admin dashboards, help center / safety center, in-app messaging beyond gathering RSVP. See [mvp-requirements.md](mvp-requirements.md#not-included-in-mvp).

**Status:** This lists deferred items already named in existing docs. A prioritized, dated roadmap has not been drafted yet.
