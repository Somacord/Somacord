# Somacord Tech Stack

## Frontend
- Next.js
- TypeScript
- Tailwind CSS

## Backend
- Supabase
- PostgreSQL

## Payments
- Stripe — Somacord Membership (monthly/annual), Community Partner Subscription (monthly/annual, a separate product), Partner Event Promotion (one-time), plus dynamic Checkout Sessions for Official Somacord Event and paid Partner Event tickets. See [/docs/engineering/integrations.md](integrations.md#stripe) and [/docs/engineering/marketplace-implementation-plan.md](marketplace-implementation-plan.md) for the full (not yet implemented) design.

## Email

- Cloudflare

## Source Control / Docs
- GitHub

## Development Environment
- Cursor

## Third-Party Experience Provider
- Meetaway: virtual conversation technology/provider powering the Speed Connect guided-conversation experience. Internal/vendor use only, never surfaced as a customer-facing product name (the customer-facing name is always "Speed Connect").

See [/docs/brand/experience-language.md](../brand/experience-language.md) for the naming rule this last point enforces.
