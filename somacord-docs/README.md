# Somacord

Somacord helps adults build real friendships through guided conversations and shared experiences — a friendship-first social club, not a dating app, social feed, or networking tool.

This repository is the product foundation for the Somacord MVP: brand, business, product, website, design, and engineering documentation, organized for development.

## Repository Guide

- **[/docs/brand](docs/brand)** — Logo, color, typography, and language standards
- **[/docs/business](docs/business)** — Business model, pricing, membership, and partner strategy
- **[/docs/product](docs/product)** — Vision, user journeys, and MVP requirements
- **[/docs/website](docs/website)** — Sitemap, SEO, and content strategy
- **[/docs/design](docs/design)** — Design system, UI components, and approved mockups
- **[/docs/engineering](docs/engineering)** — Tech stack, architecture, and database schema
- **[/assets](assets)** — Brand assets, mockup exports, and icons
